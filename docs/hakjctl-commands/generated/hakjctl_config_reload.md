## hakjctl config reload

Reload configurations

### Synopsis

Reload configurations on the HakjDB server from its configuration sources.

```
hakjctl config reload [flags]
```

### Examples

```
# Reload configurations
hakjctl config reload
```

### Options

```
  -h, --help   help for reload
```
