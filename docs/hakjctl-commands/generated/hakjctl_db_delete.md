## hakjctl db delete

Delete a database

### Synopsis

Delete the database with the specified name. Returns the name of the deleted database.

```
hakjctl db delete NAME [flags]
```

### Examples

```
# Delete database 'mydb'
hakjctl db delete mydb
```

### Options

```
  -h, --help   help for delete
```
