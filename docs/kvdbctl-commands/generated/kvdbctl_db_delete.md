## kvdbctl db delete

Delete a database

### Synopsis

Delete a database with the specified name. Returns the name of the deleted database.

```
kvdbctl db delete NAME [flags]
```

### Examples

```
# Delete database 'mydb'
kvdbctl db delete mydb
```

### Options

```
  -h, --help   help for delete
```
