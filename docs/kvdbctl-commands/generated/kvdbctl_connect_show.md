## kvdbctl connect show

Show connection settings

### Synopsis

Show the currently configured connection settings used to connect to a server.

```
kvdbctl connect show [flags]
```

### Examples

```
# Show all connection settings
kvdbctl connect show
```

### Options

```
  -h, --help   help for show
```
