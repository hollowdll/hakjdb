## kvdbctl db ls

List all the databases

### Synopsis

List the names of all the databases that exist on the server.

```
kvdbctl db ls [flags]
```

### Examples

```
# List the names of all the databases
kvdbctl db ls
```

### Options

```
  -h, --help   help for ls
```
