## kvdbctl logs

Get logs from the server

### Synopsis

Get logs from the server if the server's log file is enabled. Currently gets all the logs.

```
kvdbctl logs [flags]
```

### Examples

```
# Get all logs
kvdbctl logs
```

### Options

```
  -h, --help   help for logs
```
